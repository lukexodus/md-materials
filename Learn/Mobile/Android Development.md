# Syllabus

## Prerequisites

- Basic programming concepts (variables, functions, loops, conditionals)
- Object-oriented programming fundamentals
- Familiarity with command line/terminal basics

---

## Phase 1: Foundation (Weeks 1-4)

### Week 1: Development Environment Setup

- **Android Studio Installation & Configuration**
    
    - Installing Android Studio
    - SDK Manager and AVD Manager
    - Setting up emulators and physical devices
    - Understanding project structure
- **Version Control**
    
    - Git fundamentals
    - GitHub integration with Android Studio

### Week 2: Programming Language Foundation

**Choose one primary language:**

**Option A: Kotlin (Recommended)**

- Kotlin syntax and basics
- Variables, data types, and functions
- Classes and objects
- Null safety
- Extension functions
- Lambda expressions

**Option B: Java**

- Java syntax review
- Classes and inheritance
- Interfaces and abstract classes
- Collections framework
- Exception handling

### Week 3: Android Fundamentals

- **Android Architecture Overview**
    - Android stack and components
    - APK structure
    - Android runtime (ART)
- **Application Components**
    - Activities
    - Services
    - Broadcast Receivers
    - Content Providers

### Week 4: Basic App Structure

- **Activities and Lifecycle**
    - Activity lifecycle methods
    - Handling configuration changes
    - Intent fundamentals
- **Layouts and Views**
    - LinearLayout, RelativeLayout, ConstraintLayout
    - Common views (TextView, Button, EditText, ImageView)
    - ViewGroups and view hierarchy

---

## Phase 2: Core Development (Weeks 5-12)

### Week 5-6: User Interface Development

- **Advanced Layouts**
    - ConstraintLayout mastery
    - RecyclerView and adapters
    - CardView and custom views
- **Material Design**
    - Material Design principles
    - Material Components
    - Theming and styling

### Week 7-8: User Interaction

- **Event Handling**
    - Click listeners
    - Touch events
    - Gesture handling
- **Navigation**
    - Intent types (explicit/implicit)
    - Navigation between activities
    - Fragment basics and navigation

### Week 9-10: Data Management

- **Local Data Storage**
    - SharedPreferences
    - Internal and external storage
    - SQLite database basics
    - Room database library
- **Data Binding**
    - View binding
    - Data binding fundamentals

### Week 11-12: Advanced Components

- **Fragments**
    - Fragment lifecycle
    - Fragment transactions
    - Fragment communication
- **Services**
    - Started services
    - Bound services
    - Background processing guidelines

---

## Phase 3: Intermediate Development (Weeks 13-20)

### Week 13-14: Networking and APIs

- **HTTP Networking**
    - Retrofit library
    - JSON parsing with Gson/Moshi
    - Handling network errors
    - Authentication basics

### Week 15-16: Modern Architecture

- **MVVM Architecture**
    - Model-View-ViewModel pattern
    - LiveData and Observer pattern
    - ViewModel lifecycle awareness
- **Repository Pattern**
    - Data abstraction
    - Single source of truth

### Week 17-18: Advanced UI

- **Custom Views**
    - Creating custom components
    - Custom drawing with Canvas
    - Handling touch events in custom views
- **Animations**
    - View animations
    - Property animations
    - Motion Layout

### Week 19-20: Background Processing

- **Threading**
    - Main thread vs background threads
    - AsyncTask alternatives
    - Coroutines (Kotlin) or ExecutorService (Java)
- **WorkManager**
    - Scheduling background tasks
    - Constraints and policies

---

## Phase 4: Advanced Development (Weeks 21-28)

### Week 21-22: Advanced Architecture Patterns

- **Dependency Injection**
    - Manual DI
    - Dagger/Hilt fundamentals
- **Clean Architecture**
    - Separation of concerns
    - Use cases and domain layer

### Week 23-24: Advanced Data Management

- **Room Advanced Features**
    - Relationships and foreign keys
    - Database migrations
    - Testing Room databases
- **DataStore**
    - Preferences DataStore
    - Proto DataStore

### Week 25-26: Performance and Optimization

- **Memory Management**
    - Memory leaks prevention
    - Profiling tools
    - Bitmap optimization
- **Performance Monitoring**
    - Layout optimization
    - Network optimization
    - Battery optimization

### Week 27-28: Testing

- **Unit Testing**
    - JUnit basics
    - Mockito for mocking
    - Testing ViewModels and repositories
- **UI Testing**
    - Espresso framework
    - UI Automator
    - Testing strategies

---

## Phase 5: Professional Development (Weeks 29-36)

### Week 29-30: Security

- **App Security**
    - Secure coding practices
    - Data encryption
    - Network security
    - ProGuard/R8 obfuscation

### Week 31-32: Publishing and Distribution

- **Play Store Publishing**
    - App signing
    - Play Console setup
    - Store listing optimization
    - Release management
- **App Bundle and APK**
    - Android App Bundle
    - Dynamic delivery
    - Instant apps concepts

### Week 33-34: Modern Android Development

- **Jetpack Libraries**
    - Navigation Component
    - Paging library
    - CameraX
    - Biometric authentication
- **Jetpack Compose** (Optional but recommended)
    - Declarative UI concepts
    - Compose fundamentals
    - State management in Compose

### Week 35-36: Advanced Topics

- **Firebase Integration**
    - Authentication
    - Firestore database
    - Cloud messaging
    - Analytics
- **Machine Learning**
    - ML Kit integration
    - TensorFlow Lite basics

## Practical Projects

### Beginner Projects (Weeks 1-12):

1. **Calculator App** - Basic UI and event handling
2. **To-Do List** - Local data storage with Room
3. **Weather App** - API integration and networking

### Intermediate Projects (Weeks 13-28):

4. **Recipe App** - MVVM architecture, complex UI
5. **Expense Tracker** - Advanced data management
6. **Social Media Client** - Authentication, networking

### Advanced Projects (Weeks 29-36):

7. **E-commerce App** - Complete app with payments
8. **Chat Application** - Real-time features
9. **Portfolio App** - Publishing and optimization

- **Study Time**: 10-15 hours per week
- **Project Time**: Additional 5-10 hours per week

This syllabus provides a structured path from beginner to professional Android developer, with practical projects and real-world skills development throughout the journey.
