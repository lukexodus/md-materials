## Key Principles for Any Timeline:

**Prerequisites Flow:**
1. Math + Programming → Everything else
2. Architecture + Systems → Parallel Computing, GPU
3. Graphics Fundamentals → Advanced Graphics
4. ML Foundations → Deep Learning → Computer Vision
5. Basics → Robotics/Real-Time/HPC

**Parallel Learning Opportunities:**
- Software Engineering practices throughout
- Security concepts can be integrated alongside other modules
- Cloud computing pairs well with distributed systems

**Study Intensity:**
- 24-month: ~20-30 hours/week
- 18-month: ~30-40 hours/week
- 36-month: ~10-15 hours/week

**Adjustment Factors:**
- Prior experience can skip/accelerate certain modules
- Interest areas can determine depth of study
- Career goals may prioritize certain tracks
